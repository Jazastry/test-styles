Toggles navbar from

Used on Portal navbar to enlarge the app content to its maximum height

  1. .js--toggle-navbar${require('../_mark.html')}

    Add this class to the apps navbar to allow toggling with the navigation toggling button

  2. .js--toggle-navbar-button${require('../_mark.html')}

    Add this class to the icon of the navigation toggling button

  3. .js--toggle-navbar-parent${require('../_mark.html')}

    Add this class to the primary parent of the navigation toggling button

  4. .js--toggle-navbar-native${require('../_mark.html')}

    Applicable for native apps: Add this class to the primary parent of components the position of which has to be compensated due to visible navbar (adds top: 2.6rem)

  5. .js--toggle-navbar-composed${require('../_mark.html')}

    Applicable for composed apps: Add this class to the primary parent of components the position of which has to be compensated due to visible navbar (adds top: 9.2rem)

  6. .js--toggle-navbar-content${require('../_mark.html')}

    Needed to do the magic
