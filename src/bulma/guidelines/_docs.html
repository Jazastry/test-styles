If you're not familiar with Bulma, please find the docs here. Bulma is a framework for CSS. This work extends this with Prediktor styling.

Including it in your project

You need to download the files either for css or sass. If you want the sass-files; please find them in the original repo. The css files will include the whole bulma css library. The easiest way of adding them is as follows:

  1. In your publically available assets, make sure you have two folders: images and css. Under the css folder you need to create a mode folder. If you choose to have other names, you might need to update the contents of the files.
  2. Copy the css files to your css folder and subfolder.
    • The main file: css/prediktor-bulma.css which is responsible for including the others css's
    • The light theme file to the mode folder: css/mode/prediktor-light.css
    • The dark theme file to the mode folder: css/mode/prediktor-dark.css
    • There is standalone print theme file in mode folder: css/mode/prediktor-print.css
  3. Add a link to the css in the header of your html: <link rel="stylesheet" href="/css/prediktor-bulma.css" />
